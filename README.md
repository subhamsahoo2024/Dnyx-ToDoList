# To-Do List
A simple To-do list application for managing tasks (add, edit, complete, delete). 

## Features
- Add new tasks
- Edit existing tasks
- Mark tasks as completed / uncompleted
- Delete tasks
- Local state persistence 
- filtering completed/ Not completed tasks

## Tech stack
- HTML
- JavaScript 
- Tailwind CSS cdn

## Usage
- Add tasks using the input form.
- Click a task or a checkbox to toggle complete.
- Use edit/delete controls to modify or remove tasks.
